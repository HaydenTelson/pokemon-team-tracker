# Pokémon Team Tracker

A lightweight, browser-based Pokémon team management tool built with vanilla HTML/CSS/JavaScript.

## Features

- **Team Management**: Build and manage your Pokémon team (up to 6 members)
- **Level Tracking**: Quick level increment/decrement controls
- **Move Selection**: Choose up to 4 moves per Pokémon with detailed information
- **Team Analysis**: View type coverage and team weaknesses
- **Battle Helper**: Quick opponent lookup with super effective move recommendations
- **Research Tools**: Team gap analysis and comprehensive Pokédex viewer
- **Generation Filter**: Filter Pokémon by generation (1-9)
- **Auto-save**: All data persists in your browser's localStorage

## Live Demo

Visit the live app: [Pokémon Team Tracker](https://YOUR_USERNAME.github.io/pokemon-team-tracker/)

## Usage

Simply open the app in your browser and start building your team! All data is saved locally in your browser.

## Technologies

- Vanilla JavaScript (ES6+)
- HTML5 & CSS3
- [PokéAPI](https://pokeapi.co/) for Pokémon data
- LocalStorage for data persistence

## Local Development

1. Clone the repository
2. Open `index.html` in your browser
3. No build process or dependencies required!
